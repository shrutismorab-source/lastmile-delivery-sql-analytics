# Last-Mile Delivery Network Analytics

A SQL portfolio project simulating a last-mile delivery network, built as a
hands-on case study to learn SQL using real-world logistics domain knowledge
from my work at Amazon Logistics.

## What's inside

- SQLite database with a 5-table relational schema: stations, routes,
  drivers, packages, and delivery events
- ~288,000 synthetic packages flowing through 6 delivery stations
- 20+ queries progressing from basic SELECTs to joins, CTEs, and
  window functions
- Business questions covered: station throughput, capacity planning,
  delivery performance, route efficiency

## Why I built it

I work in Amazon Logistics operations and have 7+ years of experience in
procurement and vendor management. I wanted to add SQL to my toolkit, so
instead of following a generic tutorial, I built a dataset that mirrors the
operational world I know: stations, delivery windows, package flow.

The project was scaffolded with AI assistance (Claude) as a learning tool —
the schema and dataset were generated to give me a realistic sandbox, and I
work through every query myself following a structured study plan
(see STUDY_WALKTHROUGH).

## Tools

SQLite (sqlite3 on macOS Terminal), DB Browser for SQLite
